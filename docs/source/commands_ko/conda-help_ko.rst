conda help

   <H2>이름</H2><PRE>
          conda - conda help
   
   
   </PRE>
   <H2>설명</H2><PRE>
          사용법: conda help [-h] [COMMAND]
   
          사용 가능한 콘다 명령의 목록과 도움말을 표시한다.

   
   </PRE>
   <H2>옵션</H2><PRE>
      <B>위치</B> <B>인수:</B>
          COMMAND
                 COMMAND에 입력된 명령의 도움말 정보를 출력한다.
                 (동일한 명령: conda COMMAND <B>--help</B>).
   
      <B>선택</B> <B>인수:</B>
          <B>-h</B>, <B>--help</B>
                 도움말 메세지를 보고 종료한다.

   
   </PRE>
   <H2>예시</H2><PRE>
                 conda help install
   
   
   
   
   </PRE>
